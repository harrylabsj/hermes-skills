# Style Guide

Use this reference when shaping second-brain reflections, aphorisms, quote-card lines, article endings, or variant sets.

## Output Shapes

- `感悟`: 80-180 Chinese characters, one paragraph, reflective but concrete.
- `箴言`: 8-32 Chinese characters, one sentence or phrase, compressed and memorable.
- `金句`: 16-48 Chinese characters, slightly more public-facing than `箴言`.
- `卡片`: title plus one sentence; title should be 4-10 Chinese characters.
- `文章收束`: 1-3 sentences that close an essay without sounding final or preachy.
- `每日一念`: one sentence, quiet tone, suitable for a personal log.

## Voice

Prefer:

- plain Chinese with a little compression
- verbs that reveal motion: 看见, 放下, 练习, 承认, 穿过, 返回, 生成
- nouns that carry the user's system: 锚点, 回路, 长期问题, 经验, 选择, 注意力, 自我, 作品
- a small turn: "不是 A，而是 B" only when it sharpens the thought

Avoid:

- generic slogans such as "相信自己", "坚持就是胜利", "拥抱变化"
- overlarge absolutes such as 永远, 所有, 必须, 彻底, 真正的人生
- decorative stacking of abstract nouns
- pretending to quote a source without an actual source
- explaining the line after writing it unless the user asks

## Generation Pattern

Use this mental pattern:

```text
召回材料 -> 张力 -> 反转 -> 行动/看见 -> 压缩
```

Examples of seed transformations:

- "想要控制结果" plus "长期主义" can become "长期主义不是控制远方，而是把今天的动作做成可信的重复。"
- "第二大脑" plus "自我观察" can become "第二大脑的意义，不是替我记住一切，而是让我更快看见自己正在反复成为什么。"
- "代理系统" plus "人机协作" can become "好的代理不是替代判断，而是把判断前的混沌整理到可以选择。"

## Variant Strategy

When asked for multiple candidates, vary the underlying rhetorical move:

1. Paradox: "越是..., 越要..."
2. Reframe: "不是..., 而是..."
3. Practice: "把...变成..."
4. Observation: "你以为..., 其实..."
5. Quiet maxim: short, image-light, no preaching

For each candidate, keep one dominant idea. Do not make every version a synonym of the first.

## Final Polish

Before final output:

1. Delete any sentence that could appear in any motivational account.
2. Replace broad adjectives with the user's actual context.
3. Cut the first clause if the second clause already carries the insight.
4. Read the line aloud; if it sounds inflated, make it plainer.
