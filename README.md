# Hermes Skills By HarryLabsJ

Public Hermes-compatible skill tap maintained by harrylabsj.

## Install

Add this tap once:

```bash
hermes skills tap add harrylabsj/hermes-skills
```

Then search or install skills:

```bash
hermes skills search 京东
hermes skills install harrylabsj/hermes-skills/skills/jd-shopping
```

## Skills

| Skill | What It Does | Install |
|-------|--------------|---------|
| `jd-shopping` | JD.com search, comparison, review analysis, SKU selection, and safe cart preparation before checkout/payment. | `hermes skills install harrylabsj/hermes-skills/skills/jd-shopping` |
